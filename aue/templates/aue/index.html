Instruction Details

This would be a help system which provides links to be clicked next when the user enters his goal after providing the website URL he wishes to visit.

User interface should consist of a web page where the user can input the URL of the website he wishes to browse and also a text field to input his query (Could be a complete string of characters or keywords or a complete sentence) indicating what he wants to search in particular in the given website. The user will submit the URL and query which would process and provide the result in the form of four options in order of high to low relevance giving the next link to click and also an option to exit if he wishes to. Every link provided as a suggestion should be of the next webpage where the probability of getting closer to the required web page increases. Once the user clicks the highest probable link given in the options by the system, he is lead to the page which is one step closer to the required webpage. Hence the system should provide a path to the user for reaching the required webpage.
